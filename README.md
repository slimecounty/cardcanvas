# card canvas

AI slop - digital index-card-driven storyboard app for building stories visually and generating a traditionally formatted screenplay-style output.

**Live App:** https://slimecounty.github.io/cardcanvas/

---

## Overview

card canvas is a browser-based writing and story-planning tool built around the classic index card workflow used by screenwriters, filmmakers, and storytellers.

Instead of writing linearly from page one, card canvas lets you create, arrange, connect, and organize cards on a visual canvas. As cards are connected into a timeline, the app generates a screenplay-style story view, exportable as an html file with optional media or as plain text.

The goal is to combine the flexibility of a digital corkboard with the structure of traditional screenplay formatting.
